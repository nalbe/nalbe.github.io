  • IKeybind Library

    Arduino keybind handler built on IPushButton for multi-key combinations and event-driven input:

    • Multi-key combination support

      Define combinations of two or more physical buttons with modifier and trigger key roles. The last pin in a combination acts as the trigger while others serve as modifiers.

    • Event state detection

      Each combination can fire on specific trigger key states: release, rapid (rapid press), or hold. Enables rich input patterns from a small number of buttons.

    • Compile-time configuration

      Uses C++ templates to set the number of keys and events at compile time: IKeybind<Key_Cnt, Event_Cnt>. No dynamic allocation, ideal for resource-constrained embedded systems.

    • Seamless IPushButton integration

      Initialized with an array of IPushButton objects. Supports per-key settings like repeatDelay() via forEachKey(). Call kb.update() in loop and check events with kb.isEvent(index).
